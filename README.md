Mark CLI
=======
a cli used to learning webpack


## install

```
$ npm install mark-cli -g
```

## create 

```
$ mark init myproject
```

## dev&build

```
$ cd myproject

$ npm install

$ npm run dev //development

$ npm run build // production
```

## start

```
$ npm start
```
